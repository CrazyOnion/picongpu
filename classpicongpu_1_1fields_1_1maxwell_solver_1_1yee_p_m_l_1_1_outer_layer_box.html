Detailed Description

template<typename T_DataBox1d>
class picongpu::fields::maxwellSolver::yeePML::OuterLayerBox< T_DataBox1d >

Data box type used for PML fields in kernels.

Only stores data in the PML area using the given 1d data box. Access is provided via a simDim-dimensional index, same as for other grid values.
