Detailed Description

template<typename T_FieldSolver>
struct picongpu::fields::absorber::detail::Absorber< T_FieldSolver >

Absorber wrapper.

Provides unified interface for the absorber information: size along the 6 boundaries and getStringProperties() implementation. Currently does not provide the computational part, only description.

The general version uses exponential absorber settings since this is the default absorber.
